## CSS3有哪些新增的特性

### 边框
+ border-radius 边框圆角
+ box-shadow 边框阴影
+ border-image 边框图像

### 背景
+ background-size 背景尺寸
+ background-clip 背景图片绘制的区域
+ background-origin 背景图片的定位区域

### 渐变
+ linear-gradient 线性渐变
+ radial-gradient 径向渐变

### 文本效果
+ word-break    在恰当的断点字换行
+ word-wrap     允许长单词换行到下一行：
+ text-overflow 截断文字显示省略号
+ text-shadow   文字阴影
+ text-justify  齐行改变单词间的间隔：

### 转换：transform
+ translate() 移动
+ scale()   放大缩小
+ skew()    倾斜
+ rotate()  旋转

### 3d转换
+ transform
+ transform-origin
+ transform-style
+ 3D转换方法
+ translate3d(x,y,z)
+ translateX(x)
+ translateY(y)
+ translateZ(z)
+ scale3d(x,y,z)
+ scaleX(x)
+ scaleY(y)
+ scaleZ(z)
+ rotate3d(x,y,z,angle)
+ rotateX(x)
+ rotateY(y)
+ rotateZ(z)
+ perspective(n)

### 过渡
+ transition

### 动画
+ @keyframes
+ animation

### 媒体查询
+ @media

### 弹性盒子
+ flexbox
